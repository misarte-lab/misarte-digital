# MisArte Digital v1.5.0 — Personalização

## Instalação

1. No Supabase, abra **SQL Editor**.
2. Execute `supabase/v1.5.0-personalizacao.sql`.
3. Substitua os arquivos do repositório pelos arquivos deste pacote.
4. Faça commit e aguarde o GitHub Pages atualizar.
5. Abra um cliente no painel e acesse a aba **Aparência**.

## Recursos

- Tema claro ou escuro
- Cores personalizadas
- Fontes
- Banner
- Favicon
- Catálogo em destaque
- Prévia mobile
- Compatibilidade com a URL `publico.html?cliente=ID`

## Observação

A v1.5.0 não inclui slug nem segmentos. Esses recursos pertencem à v1.5.1.
