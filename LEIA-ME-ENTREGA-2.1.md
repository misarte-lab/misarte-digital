2. Substitua o projeto atual pelo conteúdo desta entrega cumulativa.
3. Faça o commit e o push pelo GitHub Desktop.
4. Abra a tela Aparência e pressione `Ctrl + F5`.
5. Digite um nome em **Texto de pré-visualização**.

Resultado esperado: o texto completo aparece em cada cartão usando a fonte correspondente.
